## Build Instructions ##

### Install Libraries ###
* For Debian and Debian derivatives including Ubuntu:
> sudo apt-get install libcairo2-dev libpango1.0-dev libcppunit-dev

* Others:
> Good luck!

### Download Source Code ###
> git clone https://github.com/jryu/calendar.git

### Build & run
> ./run.sh
